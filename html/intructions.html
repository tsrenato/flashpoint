How to Use?

reloads all window tabs in a determined interval time. Remember to always use seconds!
switches between all tabs that are not in the blacklist.
adds forbiden words in URLs to the Carrossel's blacklist.

Auto Reload

The Auto Reload tool have a default interval of 5 seconds for all tabs, that time can be changed to values greater than 3 seconds (avoiding unstoppable loops). It also have the Protect Current Page button wich prevent the page you are using from reloading. The tool also contains its own settings options, there you can toggle each tab you do not want to reload.

Caroussel

The Caroussel is a tool for keep switching tabs automatically given a screen time for each tab of 5 seconds by default. It is possible to change the screen time of every openned tab individually by accessing the Caroussel Settings options. Once you have setted a new screen time to a tab remember to click save to keep your custom times.

Exception URL

These are words that you can add to Caroussel's Blacklist, every word in blacklist will be checked into every tab URL, that way if the url contains the forbiden words Caroussel will then ignore that tab throught its rotation. You are able to remove words from blacklist by accessing the Remove Exceptions options.
